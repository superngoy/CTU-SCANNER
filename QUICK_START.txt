# QUICK START - Excel Student Import

## TL;DR
Yes, you can import all 26,524 student records from the Excel file to the database even though they have different attributes.

## Files Created
1. **`import_students.php`** - The import script
2. **`IMPORT_GUIDE.md`** - Complete documentation
3. **`check_excel.php`** - Excel analysis script (used for diagnosis)

## Quick Commands

### Preview (recommended first)
```
cd c:\ctu-scanner
php import_students.php preview
```
Expected output: Shows 26,524 records ready to import, 0 errors

### Import
```
cd c:\ctu-scanner
php import_students.php import
```
Then type `yes` when prompted to confirm

## What Gets Imported
| From Excel | To Database | Notes |
|---|---|---|
| idno | StudentID | ✓ Direct mapping |
| firstname | StudentFName | ✓ Direct mapping |
| lastname | StudentLName | ✓ Direct mapping |
| middlei | StudentMName | ✓ Direct mapping |
| course | Course | ✓ Direct mapping |
| year | YearLvl | ✓ Direct mapping |
| age | BirthDate | ✓ Calculated |
| *(missing)* | Gender | ⚠ Default: "Other" |
| *(missing)* | Section | ⚠ Default: "N/A" |
| *(missing)* | Department | ⚠ Default: "COTE" |
